* ex_0402_images size

```
<!DOCTYPE>
<html>
<head>
    <style>
        /* This stylesheet sets the width of all imges to 100%: */
        img{
            width: 100%;
        }
    </style>
    
</head>
    
<body>
    <h2>Styling Images</h2>
    <p>The imges below has the width attribute set to 128 pixels, but the stylesheet overrides it, and sets the width to 100%.</p>
    <img src="https://www.ksu.edu.tw/resource/image/logo/logo-og.png" width="128" height="128">
    
    <p>The images below uses the style attribute, where the width is set to 128 pixels which overrides the stylesheet:</p>
    <img src="https://www.ksu.edu.tw/resource/image/logo/logo-og.png" style="width: 128px;height: 128px;">
    
</body>
</html>
```
